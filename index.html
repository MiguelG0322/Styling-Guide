Common Styling Criteria

Anchor Tags
  • Text-decoration: None (Removes underline)
  • If the anchor tag is in the navbar: border: solid 2px black
  • Color: #05386B (Dark Blue text color)
  • Background: #5CDB95 (Mint Green Background)

Headings

  • Color: #EDF5E1 (This works great if the background is #5CDB95
  • Font-family: 'DM Sans', sans-serif
  • Font-weight: normal(400)

Paragraphs

  • Color: black
  • Font-family: 'Poppins', sans-serif
  • If important or quote: font-style: italic
  • Line-height: 1.5 (Good ratio for vertical spacing)
